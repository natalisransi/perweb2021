Belajar tentang Classes pada HTML

Atribut class ini digunakan untuk menunjuk ke nama kelas dalam CSS, hal ini digunakan untuk mempermudah styling di beberapa komponen elemen.

Belajar tentang Id pada HTML

Id ini digunakan untuk menunjukkan deklarasi style yang ada pada CSS, digukan juga untuk hyperlink ke tag tertentu, dan juga hyperlink ke link internet. Id ini hampir mirip dengan class, bedanya class dapat digunakan di beberapa elemen HTML, sedangkan Id hanya boleh digunakan oleh satu elemen HTML dalam halaman.
